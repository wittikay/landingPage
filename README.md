# My Landing Page
## Live page: https://wittikay.github.io/landingPage/

### Purpose: To consolidate projects and social links into one 'master' page.

### Technologies Used
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (vanilla)
- Google Fonts (Inter Tight)
- Normalize.css
- GitHub Pages

### Methods Used
- Responsive layout (flex/grid, clamp-based sizing)
- Image-first UI with accessible alt text
- ASCII logo animation (HTML/CSS/JS iframe embed)
- Asset compression and HTML/CSS/JS minification
- AI-assisted iteration for layout and visual polish
